## Design Workflow: From AI to Final Product

### Step-by-Step Production Pipeline

#### **Phase 1: Batch Generation (2-3 hours)**

```
1. Create prompt library (20-30 base prompts)
2. Generate variations in Midjourney (100-200 images)
3. Select top 30-40 candidates
4. Upscale favorites to 4K resolution
```

**Example Prompt Template:**

```
Seamless vintage botanical paper pattern, pressed flowers and leaves, 
aged parchment texture, muted sage green and cream colors, 
cottagecore aesthetic, high detail, commercial use --ar 1:1 --tile --style raw --v 6.1

Parameters:
--ar 1:1      = Square aspect ratio (easy to tile)
--tile        = Seamless pattern mode
--style raw   = Less artistic interpretation
--v 6.1       = Latest model version
--chaos 10    = Add variation between generations
```

#### **Phase 2: Curation & Editing (1-2 hours)**

```
1. Import to Adobe Bridge/Lightroom
2. Rate images (1-5 stars)
3. Edit top picks in Photoshop:
   - Color correction
   - Remove artifacts
   - Enhance textures
   - Fix tiling seams
4. Export as PNG (300 DPI, sRGB)
```

**Photoshop Actions to Create:**
- Auto color correction
- Noise reduction
- Sharpening for print
- Batch resize to standard sizes

#### **Phase 3: Product Assembly (30 min per pack)**

```
Standard Pack Contents:
├── 12 unique designs (coordinated theme)
├── 3 size variants each:
│   ├── 8.5" x 11" (US Letter)
│   ├── A4 (210 x 297 mm)
│   └── 12" x 12" (Scrapbook standard)
├── Preview image (collage of all designs)
├── Thank you card / instructions
└── ZIP file with organized folders
```

**Folder Structure:**
```

├── A4/

│   └── ...
└── 12x12/
    ├── design_01.png
    └── ...
```

#### **Phase 4: Quality Control (15 min per pack)**

```
Checklist:
□ All files open correctly
□ Resolution is 300 DPI minimum
□ Colors are accurate (print test)
□ No visible tiling seams
□ File names are clear
□ ZIP extracts without errors
□ Preview image is attractive
```

### 5. Price Optimization Bot

```python
# Monitor competitor prices and adjust
import requests
from bs4 import BeautifulSoup

def scrape_competitor_prices(search_term):
    url = f'https://www.etsy.com/search?q={search_term}'
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')
    
    prices = []
    for price_element in soup.select('.currency-value'):
        prices.append(float(price_element.text))
    
    avg_price = sum(prices) / len(prices)
    return avg_price

def adjust_price(product_id, competitor_avg):
    # Price 5% below average
    new_price = competitor_avg * 0.95
    
    etsy.update_listing(product_id, {'price': new_price})
```

**Important:**
- Always check current Terms of Service
- Keep records of subscriptions (proof of commercial license)
- Some platforms require paid plans for commercial use

### Etsy Policies

**Allowed:**
- ✅ AI-generated art (must disclose if asked)
- ✅ Digital downloads
- ✅ Commercial use licenses to buyers

**Not Allowed:**
- ❌ Copyrighted characters (Disney, anime, etc.)
- ❌ Trademarked logos or brands
- ❌ AI art claiming to be "hand-painted"
- ❌ Reselling others' AI outputs

### Copyright Considerations

**What You Own:**
- ✅ Your curated selections
- ✅ Your edits and modifications
- ✅ Your compilations and arrangements
- ⚠️ AI outputs (legally gray, varies by country)

**Best Practices:**
1. Make significant edits in Photoshop
2. Combine multiple AI outputs
3. Add original elements (text, graphics)
4. Don't claim "hand-made" if AI-generated
5. Be honest if customers ask

### Tax Implications

**US Sellers:**
- Report income on Schedule C
- Self-employment tax applies
- Keep expense records (subscriptions, software)
- 1099-K issued if >$600/year (2026 threshold)

**International:**
- VAT may apply (EU digital products)
- Check local regulations
- Etsy collects VAT in some regions
